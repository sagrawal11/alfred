Testing Instructions

  1. Check Scheduler Status: Verify the scheduler is running
  2. View Scheduled Jobs: See all configured background jobs and their next run times
  3. Test Reminder Follow-ups:
    • First, create a reminder via chat: "remind me to test in 1 minute"
    • Wait for the reminder to be sent
    • Don't respond to it
    • Click "Check Reminder Follow-ups" - you should see a follow-up message in chat
  4. Test Task Decay:
    • Create an old todo (or modify database to make it 7+ days old)
    • Click "Check Task Decay" - you should see a message asking if you still want the task
  5. Test Gentle Nudges: Click to check if you need water/gym nudges
  6. Test Weekly Digest: Click to send yourself a weekly summary
  7. Test Integration Sync: Click to sync all active integrations

Note: For jobs that send messages, check the Chat Interface to see responses (instead of SMS).
